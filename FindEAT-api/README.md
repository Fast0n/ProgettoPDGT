# FindEAT-api

## Methods
#### search
Search restaurants from location

Parameters:
- lista [required] - place the position 'lat','lng'


Example Usage:
```
findeat-api.herokuapp.com/?lista=43.7292325,12.6129107
```
Returns
The standard JSON array

- name
- adress
- location
- opening
- telephone number
- website
- rating
- times
